Vrije software

Vrije software is de verzamelnaam van software die tegenwoordig voorla verspreidt onder                                             GNU General Publoc License (GPL).

Open source

Open source is een term die eind jaren negentig is bedacht om gebruik van vrije software door bedrijven te stimuleren. Door sterk de nadruk te leggen op rechten van gebruikers en het vrijgeven van zoveel mogelijke broncode, hadden veel bedrijven de indruk dat zij geen gebruik konden maken van vrije software.

Misvattingen

Vrije software mag wel verkocht worden. Er bestaat dus, in principe althans, wel degelijke commerciële vrije software. Al is dat wel moeilijk vol te houden: elke koper van zulke vrije software mag immers zelf de software begginen te kopiëren en verder verhandelen, wat natuurlijk tot een neerwaartse druk op de brijs leidt.
